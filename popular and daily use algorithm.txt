2]Quick Sort: Efficient sorting algorithm with average-case time complexity of O(n log n).
uses:-
Sorting a large dataset of numbers or strings quickly.
Organizing a list of names in alphabetical order.

3]Merge Sort: Stable sorting algorithm with guaranteed O(n log n) performance.
Merging two or more sorted lists into one sorted list.
Sorting data that is too large to fit into memory (external sorting).

4]Binary Search: Efficient search algorithm for finding an element in a sorted array.
Searching for a specific value in a sorted array.
Finding an item in a telephone directory.

7]Breadth-First Search (BFS): Traversal algorithm for exploring nodes level by level in a graph.

8]Depth-First Search (DFS): Traversal algorithm for exploring nodes deeply before backtracking.
Solving puzzles such as mazes by exploring all possible paths.
Performing a topological sort of a directed acyclic graph.
